*** Installation of cheat.mnu ***
NOTE: this menu is only usable on the test/beta server (justin)

Copy the cheat.mnu file to your CoH installation folder\data\texts\English\menus\
Example: C:\City of Heroes\data\texts\English\menus\
create any folder that is missing

in-game, paste the following into the chat
/macro_image "InherentBase_Book" "SKKP Cheat Menu" "popmenu cheats"

Happy Testing!
